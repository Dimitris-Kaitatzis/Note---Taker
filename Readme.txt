1. Unzip File
2. Copy "ToolsApp" Folder to "C:" Path
3. Run setup from "Note-Taker-main\Note Taker\Debug"

If Windows Defender Blocks the Application do theese steps: 
--> Settings 
--> Privacy And Security 
--> Windows Security 
--> Virous & Threat protection 
--> Management Settings 
--> Add or Remove Exception 
--> Add exception(choose folder option) 
--> Choose the Folder Path of the application


You can save your notes on your local Data Base. You can also use the "notepad" option to create and edit Notes. Important hint! If you want to change font size, you can press ctrl + Up key to increase the size, or ctrl + Down key to reduce the size.
